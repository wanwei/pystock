### 可用策略

| 策略类 | 功能描述 |
|--------|----------|
| `HighBreakoutStrategy` | 高点突破策略 - 价格突破过去一段时间的高点 |
| `MACrossStrategy` | 均线交叉策略 - 短期均线上穿/下穿长期均线 |
| `MACDStrategy` | MACD策略 - MACD金叉/死叉信号 |
| `RSIStrategy` | RSI策略 - 超卖/超买区域信号 |
| `CompositeStrategy` | 组合策略 - 多策略投票机制 |

### 扫描执行流程

```python
def start_scan(self):
    # 1. 获取筛选后的股票列表
    stocks = self.filter_widget.get_filtered_stocks()
    
    # 2. 获取策略和参数
    strategy_class = self.get_selected_strategy()
    params = self.get_strategy_params()
    strategy = strategy_class(**params)
    
    # 3. 获取过滤条件
    filters = self.get_scan_filters()
    
    # 4. 执行扫描
    scanner = StockScanner()
    results = scanner.scan_latest(
        strategy=strategy,
        filters=filters,
        show_progress=True
    )
    
    # 5. 显示结果
    self.display_results(results)
```
