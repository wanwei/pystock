# 股票搜索与筛选功能 - 开发清单

## 功能概述

实现**两阶段筛选流程**：

1. **第一阶段 - 基础筛选**：通过代码/名称/板块/市场等条件过滤出股票范围
2. **第二阶段 - 策略筛选**：在筛选结果上应用策略扫描，找出符合条件的股票

## 技术方案

### 第一阶段实现

```python
# 筛选条件数据结构
filter_conditions = {
    'search': '',           # 搜索关键词
    'sectors': [],          # 选中的板块列表
    'markets': ['A股'],      # 选中的市场列表
}

# 筛选流程
def apply_filters(stocks, conditions):
    result = stocks
    if conditions['search']:
        result = filter_by_search(result, conditions['search'])
    if conditions['sectors']:
        result = filter_by_sectors(result, conditions['sectors'])
    if conditions['markets']:
        result = filter_by_markets(result, conditions['markets'])
    return result
```

### 第二阶段实现

```python
# 集成StockScanner
from strategy.scanner import StockScanner
from strategy.strategies import HighBreakoutStrategy

# 在筛选结果上执行策略
scanner = StockScanner()
strategy = HighBreakoutStrategy(**params)
results = scanner.scan_latest(
    strategy, 
    filters={'min_volume': 1000000, 'min_price': 5},
    show_progress=True
)
```
